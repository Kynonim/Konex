# Konex (Vibe Coding)

## Client (Frontend SolidJS)
```bash
cd client
bun install
bun start
```

## Server (Backend ActixWeb)
```bash
cd server
cargo run --release
```

## Environment (Backend .env)
```bash
DATABASE_URL = postgres://superuser_name:superuser_password@localhost:port/database_name
JWT_SECRET = your_jwt_secret
SERVER_HOST = your_server_host
SERVER_PORT = your_server_port
```
